#  Projet Machine Learning - Prédiction de survie du Titanic

##  Description
Projet réalisé dans le cadre du cours de **Machine Learning** à la **Faculté des Sciences Semlalia – Marrakech (FSSM)**.

L'objectif est de prédire si un passager du Titanic a survécu (classification binaire) en comparant 6 algorithmes de ML.

## Objectifs
- Nettoyage et préparation des données
- Feature engineering
- Comparaison de 6 modèles
- Identification du meilleur modèle

##  Résultats
| Modèle | Exactitude (Test) |
|--------|-------------------|
| **Arbre de Décision** | **83.24%**  |
| Gradient Boosting | 82.68% |
| SVM | 82.68% |
| Régression Logistique | 82.12% |
| KNN | 82.12% |
| Forêt Aléatoire | 81.01% |

##  Technologies
- Python 3.10
- Google Colab
- pandas, numpy, matplotlib, seaborn, scikit-learn
